# bash:/practical/stable_diffusion
Practical Code Examples for Stable Diffusion

## Prompting Stable Diffusion with Bash scripts on Raspberry Pi

We will install Stable Diffusion on Raspberry Pi, generate an image generation prompt with a Bash script, and control the image generation AI using the Web UI API.  

Raspberry Pi 上から Bashスクリプトで Stable Diffusion を制御します。
